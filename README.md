⚙️ **Installation**

Clone the repository:
```bash
git clone <your-repo-url>
cd ml-ai-portfolio-projects
```
Install dependencies:
```bash
pip install -r requirements.txt
```
---

▶️ **Run Individual Project**
```bash
Example:
cd 01_healthcare_readmission_risk
python train.py
🚀 Run All Projects
python run_all.py
```
